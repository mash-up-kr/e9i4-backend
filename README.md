# Boilerplate_TypeScript_MySQL

> Boiler project with TypeScript and MySQL



## 🖥  Installation

```bash
# Install package
$ npm install

# Run prod mode
$ npm run start

# Run dev mode
$ npm run dev
```



## 🛠 Technologies

- Node.js
- MySQL
- TypeScript
